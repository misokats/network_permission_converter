# network_permission_converter
復習課題：ネットワーク・パーミッション変換器

【問題】
以下の3つの小問を、ひとつのスクリプト内で解いてみてください。

1,文字列の分解と結合

変数 path = "/usr/local/bin/python3" があります。これを split で分割し、最後だけ join を使って usr-local-bin という文字列を作ってください。

2,基数変換と0埋め（8進数編）

Linuxの権限（パーミッション）で使われる「7（読み書き実行）」などの数字を2進数に直します。

数値 7 を bin() で2進数にし、[2:] と zfill(3) を使って 111 という3桁の文字列にしてください。

3,リスト内包表記の実践

数値のリスト ports = [80, 443, 22, 8080] があります。

リスト内包表記を使って、すべての数値を文字列（str）に変換した新しいリスト str_ports を作ってください。

【ボーナス問題：これまでの集大成】
ユーザーから 255.255.0.0 というサブネットマスクが入力されたら、/16 という形式（CIDR表記）に変換して表示する関数を作ってみてください。

ヒント:

各数字を bin().zfill(8) で2進数にする。

全体をひとつの長い文字列に合体させる。

その文字列の中に 1 がいくつ含まれているか .count('1') で数える。
